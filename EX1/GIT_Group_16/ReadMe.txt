קובץ Read Me – הוראות שימוש במחשבון המרה בין בסיסים 
שלום לכולם, 
הריצו את התוכנית באמצעות תוכנה להרצת JAVA.
על מנת להשתמש במחשבון המרות בין בסיסים אנא עקבו אחר הפעולות הבאות: 
באפשרותכם להמיר מספר מבסיס עשרוני (10) לבסיס בינארי (2) ולהיפך. 
ראשית בתפריט הבחירה בחרו בבסיס ממנו תרצו להתחיל בהמרה, המחשבון ימיר באופן אוטומטי לבסיס השני. 
בחרו 1 אם תרצו להמיר מבסיס בינארי לבסיס עשרוני
בחרו 2 אם תרצו להמיר מבסיס עשרוני לבסיס בינארי 
בחרו 0 אם ברצונכם לסיים את השימוש במחשבון
לאחר מכן תוכלו להזין את המספר הרצוי ולקבל את המספר שהוזן לאחר ההמרה. 
שימו לב
- ניתן להזין מספרים שלמים חיוביים בלבד
-את המספרים יש להקליד באופן רציף ללא רווחים
לבסוף יופיע תפריט בחירה האם תרצה להמיר מספר פעם נוספת או לסיים את המשחק.
